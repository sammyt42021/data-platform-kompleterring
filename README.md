# Product Finder ETL Project

This project follows a clear ETL flow for my komplettering assignment:
1. **Extract**: generate and read product data from CSV
2. **Transform**: clean/validate data and create statistics
3. **Load**: save results to PostgreSQL and optionally stream via Kafka


## Tech Stack
- Python
- PostgreSQL
- pgAdmin
- Kafka
- Docker Compose
